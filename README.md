# 💌 Love Proposal Logger

A simple and fun **Love Proposal Logger** that helps you record love proposals and finally get a **clear answer**.

---

## 😂 Problem Statement

Love proposals are risky…

You ask:
> "Do you like me?"

And you get answers like:

- "Let's see..."
- "We'll talk later..."
- "I'm not ready..."
- "You're a good friend..."
- "Maybe..."

After all that… you still don't know if it's **Yes** or **No** 😭

This **Love Proposal Logger** helps you **log proposals and track the actual responses** so you don't get lost in confusing answers anymore.

Because sometimes...  
**Love needs documentation too.** 📋💔❤️

---

## ✨ Features

- 💌 Log love proposals  
- 📝 Record receiver responses  
- 📅 Track proposal dates  
- 📂 View saved proposals  
- ⚡ Lightweight and simple UI  
- 🌐 Runs directly in browser
